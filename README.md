# air-tickets-web
Web client for searching and comparing flight tickets with filters by price, date, and destination. Built with React.

### Features
- Search flights by destination and date
- Price and airline filters
- Flight details page
- Destination images
- Form validation and reset

### Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- React Router
